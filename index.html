Java: A Functional Approach to Java

El Mas Reciente de Todos


Se decidio escribir este libro para resaltar los diferentes conceptos que hacen un lenguaje funcional, y cómo puede incorporarlos en su Java código, ya sea con las herramientas proporcionadas por el JDK o creándolos usted mismo.

Lo más probable es que un enfoque funcional de su código Java desafiar el status quo e ir en contra de las mejores prácticas que estaba usando antes de.

Pero al adoptar una forma más funcional de hacer las cosas, como inmutabilidad y funciones puras, podrá escribir de forma más concisa, código más razonable y preparado para el futuro que es menos propenso a errores.

Enfocado A Personas:

Este libro es para ti si tienes curiosidad acerca de la programación funcional y quieres saber de qué se trata todo este alboroto y aplicarlo a tu código Java.

Es posible que ya esté utilizando algunos tipos de Java funcionales, pero desea un conocimiento más profundo de por qué y cómo aplicarlos de manera más efectiva.

No es necesario ser un experto en programación orientada a objetos, pero el libro no es una guía para principiantes de Java o programación orientada a objetos.

Ya debería estar familiarizado con la biblioteca estándar de Java. No se requieren conocimientos previos de programación funcional. Cada concepto se introduce con una explicación y ejemplos.
